# CALMA - Sistema de Gestión de Tareas

CALMA es una plataforma web de gestión de tareas diseñada para mejorar la organización personal y el seguimiento de actividades mediante una interfaz intuitiva, control de usuarios y herramientas administrativas.

El proyecto fue desarrollado como una solución práctica para la gestión de tareas, incorporando funcionalidades comunes en aplicaciones SaaS modernas como autenticación, control de roles y panel administrativo.

---

## Características Principales

✅ Registro de usuarios

✅ Inicio de sesión seguro

✅ Gestión de perfiles

✅ Control de roles y permisos

✅ Dashboard administrativo

✅ CRUD completo de tareas

✅ CRUD de usuarios

✅ Organización de actividades por estado

✅ Interfaz responsiva

✅ Arquitectura basada en APIs REST
